EOSIDE v${version} New Release Notes

Compatibility with EOSFactory version 3.3.0 and higher

The newest EOSIDE features depend on the changes introduced in the EOSFactory version 3.3.0. In order to use those features, update EOSFactory using: 

pip install eosfactory-tokenika

EOSIDE installation view has been updated. Missing and outdated dependencies are now clearly indicated, automatic fix (if possible) is offered. See this video for more information on the new installation view.

Video gallery

Videos chapter has been added to the EOSIDE documentation. This gallery will include How-To video guides that will help you to understand EOSIO smart-contract technology and the EOSIDE instructions. We will update this gallery frequently with additional useful content. Any suggestions are highly appreciated.

New Release Notes

The new EOSIDE installation view is automatically displayed during the EOSIDE update. In order to display this view again, please use: ctrl/cmd+shift+p EOSIDE: release notes.
