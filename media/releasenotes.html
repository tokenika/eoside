EOSFactory version 3.2.0

New features of EOSIDE depend on changes introduced to EOSFactory. Please update it: 

pip install eosfactory-tokenika

Native unit tests and debugging of EOSIO smart contracts

Contract Development Toolkit supports native compilation of contracts and offers a unit test API, as it is briefly explained there EOSIDE encapsulates these processes. Also, it facilitates native debugging of EOSIO contracts.

Please, see a short video that shows EOSIDE in action, explaining the basis of the unit test API.

Packaging utility for EOSIO smart contract projects

Usually, a project contains binaries and private data and therefore it has to be processed prior to distribution. EOSFactory has a tool for automatization of of the processing.

EOSIDE implements the EOSFactory tool with the Zip (ctrl/cmd+alt+z) button in the Setup view -- for compression, and Open Zip file... in the EOSIDE view -- for creating a new project from a zip file.

See a tutorial on the issue.
