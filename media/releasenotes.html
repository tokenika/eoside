EOSFactory version 3.3.0

New features of EOSIDE depend on changes introduced to EOSFactory. Please update it: 

pip install eosfactory-tokenika

New features is better user interface of the view Installing EOSIDE. There, missing or outdated dependencies are clearly indicated, and automatic fix is offered. See a video.

Video gallery

There is a new chapter in the documentation of EOSIDE, named Videos Its contents will grow. We are going to produce easy-to-consume instructions to various aspects of the EOSIO smart-contract technology. We call for suggestions of interesting issues.
